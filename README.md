# Finding the Best Budget-Friendly Travel Destinations

Group project for **COMP 541 – Data Mining** (CSUN). We use data mining techniques to identify and rank the best budget-friendly travel destinations worldwide, based on cost of living, weather, and traveler ratings (TripAdvisor, OpenWeatherMap, and cost datasets).

## Project Phases

- **Phase 1 & 2** – Gather Data & Exploration (data collection, EDA)
- **Phase 3** – Data Preprocessing, Part 1 (cleaning, merging)
- **Phase 4** – Data Preprocessing, Part 2 (feature selection & extraction)
- **Phase 5** – Modeling (clustering and ranking destinations)

## Key Files

- `Final Project Report.docx` – full project report
- `Finding the Best Budget-Friendly Travel Destinations.pptx` – presentation slides
- `travel_dashboard.html` – interactive demo dashboard of results

## Tools

Python (pandas, scikit-learn), Jupyter Notebook
